# AKR-projekt

Instalace - Linux:
------------------
Posloupnost příkazů:\
-`git clone https://github.com/BorekZnovustvoritel/AKR-projekt`\
-`cd AKR-projekt`\
-`python3 -m venv ./venv`\
-`source venv/bin/activate`\
-`python3 -m pip install -r requirements.txt`\
-`python3 fact_proj.py`

Instalace - Windows:
--------------------
Posloupnost příkazů:\
-`git clone https://github.com/BorekZnovustvoritel/AKR-projekt`\
-`cd AKR-projekt`\
-`py -m venv ./venv`\
-`venv\Scripts\activate`\
-`py -m pip install -r requirements.txt`\
-`py fact_proj.py`
